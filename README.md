###Installation
1. Clone the repo to disk
`git clone https://github.com/sheshbabu/Tamatar.git`

2. Open Chrome -> Menu -> More Tools -> Extensions

3. Enable `Developer mode`

4. Click `Load unpacked extension...` and select the repo directory
